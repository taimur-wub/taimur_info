Research & Projects

My research integrates structural engineering with modern machine learning to develop interpretable, reliable models for seismic response, structural dynamics, and advanced cementitious materials. The focus is on scalable data generation, ensemble learning, probabilistic modeling, and symbolic regression that lead to practical tools, dashboards, and design equations for engineers.

1. Data Driven Fundamental Period Models for Steel Braced RC Frames

A comprehensive research program that constructs large scale numerical datasets and machine learning models to predict the fundamental period of steel braced RC buildings more accurately than traditional empirical code expressions. Using automated ETABS based model generation, stacked generalization, model interpretability, and deep symbolic regression, this work captures the real influence of bracing systems, geometry, and stiffness distributions.

Why this research is valuable

  • Existing code formulas are often calibrated on limited data and primarily depend on building height.
  • Braced RC frames are widely used in practice, but their dynamic characteristics are poorly represented in conventional period expressions.
  • This project provides data driven and interpretable models that reduce uncertainty in seismic demand estimation and support performance based design.

Key methods and contributions

  • Automated generation and eigenvalue analysis of 17,280 braced RC frame models using the ETABS API.
  • Stacked generalization ensembles with optimized base learners to achieve high accuracy and robustness.
  • SHAP based interpretability to identify how individual parameters (height, brace type, configuration, mass and stiffness distribution) influence the fundamental period.
  • Deep symbolic regression to derive closed form, code compatible period formulas for different bracing systems, linking ML outputs back to engineering intuition.
  • Supported by the World University of Bangladesh Internal Research Grant.

Key publications

2. Bayesian Optimized LightGBM for Masonry Infilled RC Frames

This project develops a LightGBM based model, tuned via Bayesian optimization, for predicting the fundamental vibrational period of masonry infilled RC frames. The emphasis is on speed, stability, and deployment readiness, enabling engineers to update predictions rapidly as new data or design scenarios emerge.

Why this research is valuable

  • Masonry infills significantly modify lateral stiffness yet are often neglected or oversimplified in dynamic modeling.
  • Fast and accurate ML models make it feasible to perform rapid what if analyses during retrofit, assessment, and code calibration.
  • The approach demonstrates how production quality ML can be integrated in structural engineering workflows rather than used as standalone experiments.

Key methods and contributions

  • Bayesian optimization to efficiently identify LightGBM hyperparameters for structured engineering data.
  • Model architecture designed for rapid retraining and sensitivity exploration.
  • Interactive dashboard implementation for practical use by structural engineers.

Key publication

  • Bayesian optimized LightGBM model for predicting the fundamental vibrational period of masonry infilled RC frames (2024) (Frontiers of Structural and Civil Engineering, Scopus Q1)

This work also marks my first joint publication with Shamima Sultana, reflecting a personal and collaborative milestone in structural ML research.

3. NGBoost with DSMBO for UHPC Compressive Strength Prediction

A probabilistic machine learning framework for predicting the compressive strength of Ultra High Performance Concrete (UHPC) using heterogeneous datasets collected from multiple studies. The model integrates Natural Gradient Boosting (NGBoost) with dynamic sequential model based optimization, enabling accurate, uncertainty aware predictions that generalize across variations in mix design, materials, and testing conditions.

Why this research is valuable

  • UHPC design relies on expensive experiments and scattered datasets with inconsistent distributions.
  • Conventional ML models often show reduced reliability when trained on merged heterogeneous data.
  • This framework improves predictive stability while quantifying uncertainty, supporting more confident mix design and performance evaluation.

Key methods and contributions

  • Compiled 920 UHPC mixes with 14 input parameters from diverse experimental sources.
  • Used NGBoost with DSMBO to adapt hyperparameters to heterogeneous data landscapes.
  • Applied SHAP based analysis to identify influential constituents and provide interpretable insights for material engineers.
  • Developed a web based GUI for real time UHPC strength prediction and scenario testing.

Key publication

  • Optimizing NGBoost with dynamic sequential model based optimization for predicting UHPC compressive strength on heterogeneous datasets (2025) (Materials Today Communications, Scopus Q1)
